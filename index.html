Me

About

My name is Ethan Ngawaka, I am a young junior software engineer in my second year of an honours in Computer Science at the University of Technology Sydney (UTS).

I fell in love with programming 8 years ago, making small games and prototypes in Python using Pygame. I have never lost my passion for game development but have since branched out into different areas of software development including simple full stack web development and desktop application development.

I love breaking down and understanding systems which is why I've been so drawn to game development as it covers so many challenging aspects of software. User input, satisfying feedback, general ease of use, aesthetics, and complex interlocking systems which are fundamental to most games keep me coming back again and again.

I also have a passion for mathematics and physics, studying as much as I can in my spare time. This love for maths earned me various awards during my last years of highschool including the Edward Hutley Bursary for Maths award in 2022.


Below are a series of projects completed for myself or a freelance client each showcasing; some images from the project, a list of technologies, a retrospective/reflection, and a link to source code or a place to view the project.

Personal Projects

Untitled Roguelike

Python, Pygame, GLSL

This is by far my largest project to date and the one I am most proud of. The game showcases many complex systems working together in harmony, being an incremental roguelike with upgrades, abilities, shops, waves, and unique enemies and mini-bosses. This was also my first game using Pygame in a long time, and I feel even in it's unfinished state I've almost pushed it to it's limits with a handbuilt UI system, scene managing, and shaders being just a few examples of this.

C++ Game Engine

C++, GLEW, GLUT, OpenGL, GLSL, CMake

This was my first foray into C++ and the technologies surrounding it. I ended up making a (looking back, quite terrible) game engine that handled 3D physics, vertex rendering with OpenGL, shaders, and mesh loading and handling. This was probably the most challenging experience by far, while very rewarding I should have learned the fundamentals of C++ before trying this.

The Abyss

Javscript, HTML5, Canvas, CSS

Created in 48 hours for the MicroJam 021, this won first place. This was my first experiment with procedural animation and a slight horror aesthetic. While I think it turned out well for such a short time frame, I want to explore procedural animation more.

Gallery Watch

Javscript, HTML5, Canvas, CSS

Gallery watch was created with a friend of mine for a game jam of a youtuber we enjoyed. We both did everything together and this project was my real first introduction to using git and source control while working on a team.

Growth Factor

Javscript, HTML5, Canvas, CSS

Growth Factor was a bit of a dissapointing one, I focused way too much on making the shooting feel good and aesthetics of the game that it ended up feeling pretty bad to play. That being said, having experimented now with player feedback like that, I can easily reimplement it in other contexts.

Magic Boomerangs

Javscript, HTML5, Canvas, CSS

Magic boomerangs was one of my first "fun" games, I had been obsessed with this one mechanic of throwing boomerangs and swinging them around and I took that and ran with it. There are plenty of flaws with the surrounding systems but every time I come back I can play it for way longer than any other game I've made.

Light Hunter

Javscript, HTML5, Canvas, CSS

This game was my attempt at a boss rush type game as I had played something similar recently and watned to give it a try myself. For a 48 hour jam I think I did well, I wish I had focused more on variety of attacks.

Self Destruct

Javscript, HTML5, Canvas, CSS

Self Destruct was made for a game jam with a limitation in color palette and resolution to match that of an old nokia phone. I enjoyed working around the limitation and focusing on such a simple game mechanic made it easy to shine, with a bit more time and polish this would make a really good phone time waster.

Revolve

Python, Pygame

One of my first games I uploaded on itch.io that was completed for a game jam, it won 2nd place and I think it is one of my more aesthetically pleasing games, it isn't very fun. All things considered, it is nice to have this documented to see how far I've come since.
